Цифры

Вычислить количество цифр целого неотрицательного числа n.

Входные данные

Одно целое неотрицательное число 0 <= n <=2*10**9.

Выходные данные

Выведите количество цифр в числе n.

Входные данные #1

12343

Выходные данные #1

5

Входные данные #2

985454

Выходные данные #2

6
